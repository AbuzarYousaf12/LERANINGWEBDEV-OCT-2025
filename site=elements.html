HTML Entities

HTML entities are used to display reserved characters in HTML or to display characters that cannot be typed directly.

For example:

  • The less-than sign (<) is written as &lt;
  • The greater-than sign (>) is written as &gt;
  • The ampersand (&) is written as &amp;
  • The double quote (") is written as &quot;
  • The single quote (') is written as &#39;

Here is an example of using HTML entities:

5 < 10 and 10 > 5

Use &nbsp; for a non-breaking space: This is an example.
